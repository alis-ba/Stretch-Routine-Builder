# Stretch Routine Builder

## Overview

Stretch Routine Builder is a simple web application that generates stretching routines based on user-selected mobility goals. It helps users perform quick and effective stretching exercises with a built-in timer.

## Features

* Generate personalized stretch routines
* Multiple mobility categories

  * Hip Mobility
  * Neck Relief
  * Shoulder Mobility
  * Full Body
* Interactive user interface
* Built-in countdown timer
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript
